# ▶ FLASH ANZAN　フラッシュ暗算

数字が次々と画面に表示されるのを見ながら、頭の中で計算するトレーニングアプリです。  
インストール不要・登録不要。ブラウザだけで今すぐ遊べます。

## 🔗 アクセス

> https://nadalbrian045-afk.github.io/flash-anzan/

## どんなアプリか

フラッシュ暗算とは、瞬時に表示される数字を暗算で合計していくトレーニングです。  
そろばん学習や脳トレとして広く使われています。

このアプリでは、設定した口数・桁数・スピードで数字が次々とフラッシュ表示され、最後に答えを入力して正誤を確認します。

---

## 画面の流れ

```
設定画面 → 設定確認 → カウントダウン → フラッシュ表示 → 答え入力 → 結果表示
                                                          ↑_________↓（複数問の場合）
```

1. **設定画面** — 口数・桁数・スピードなどを設定してスタート
2. **設定確認（2秒）** — 桁数・口数・合計秒数を大きく表示して確認
3. **カウントダウン** — 円形のプログレスバーでスタートを予告（ビープ音あり）
4. **フラッシュ表示** — 数字が1つずつ瞬時に表示される（表示のたびに効果音）
5. **答え入力** — 合計を入力して「決定」
6. **結果表示** — ○/× と正解の数字を表示。次の問題へ進むか設定に戻るかを選択

---

## 設定項目

| 項目 | 範囲 | 説明 |
|------|------|------|
| **口数（個）** | 2〜30 | 1問で表示する数字の個数 |
| **桁数** | 1〜4 | 各数字の桁数（例：2桁なら10〜99） |
| **合計表示時間（秒）** | 0.10〜99.99 | 全数字の表示にかける合計時間。口数で割った値が1つの数字の表示時間になる |
| **問題数** | 1〜99 | 1セッションで連続出題する問題数。複数問の場合は「第N/M問」と進捗表示 |
| **演算モード** | 加算のみ / 加減算 | 加算のみ：すべて足し算。加減算：引き算も混在（答えが0以上になるよう自動調整） |

### 設定のコツ

- **初めての方**：口数5・桁数1・合計3秒あたりからスタートするのがおすすめ
- **スピード重視**：合計時間を短くすると1つあたりの表示時間が短くなる
  - 例：口数10・合計2秒 → 1つあたり0.2秒表示
- **難易度を上げる**：口数を増やす → 桁数を増やす → 表示時間を短くする、の順がおすすめ

## 操作方法

### PC（キーボードあり）
- 答えを入力欄に直接タイプ
- **Enter キー** で決定・次の問題へ進む
- 結果画面では **F1 / F2 / F3 キー** でボタン操作が可能
  - F1：同じ問題をもう一度
  - F2：同じ設定で新しい問題
  - F3：設定を変える

### スマホ・タブレット
- 画面下部に**テンキー**が表示される（システムキーボードは起動しない）
- 数字ボタンをタップして入力 → **「決定」** ボタンで確定
- **⌫** ボタンで1文字削除
- **−** ボタンでマイナス符号のオン/オフ（加減算モード時に使用）
- 5桁以上の答えになると、入力欄の文字が自動的に小さくなって枠に収まる

## 技術的な特徴

- **単一ファイル構成** — `index.html` 1ファイルで完結。サーバー不要でローカルでも動作
- **インストール不要** — ブラウザで開くだけ。アカウント登録・ログイン一切なし
- **オフライン対応** — 一度読み込めばネット接続なしでも使用可能
- **PWA対応** — スマホのホーム画面に追加するとアプリのように起動（iOS / Android）
- **カスタムフォント** — そろばん表示をイメージしたオリジナルフォント「Abacus2」を使用
- **効果音** — Web Audio API によるビープ音（外部ファイル不要）
- **レスポンシブ対応** — PC・スマホ・タブレットいずれでも最適なレイアウトで表示
